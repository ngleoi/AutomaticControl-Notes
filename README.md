# Automatic Control Notes — 自动控制原理（第八版）学习笔记

> **胡寿松、姜斌、张绍杰 主编 · 科学出版社**

本仓库为《自动控制原理（第八版）》的个人学习笔记，包含教材全文 OCR 整理、概念梳理、公式推导、习题解答与示意图。

## 笔记特点

- **公式可读**：使用 $\LaTeX$ 数学公式书写，兼容 GitHub 渲染
- **图文并茂**：每章配有高清扫描图片及结构图
- **结构清晰**：按教材章节组织，附带习题
- **持续更新**：逐步完善各章节内容

## 使用方式

1. 克隆本仓库：
   ```bash
   git clone https://github.com/ngleoi/AutomaticControl-Notes.git
   ```
2. 使用支持 $\LaTeX$ 的 Markdown 编辑器（如 Typora、VS Code + Markdown Preview Enhanced）获得最佳阅读体验
3. 图片存放在 `images/` 目录下，按章分类
